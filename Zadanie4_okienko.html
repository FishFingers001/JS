1)kiedy powstał JavaScript (podaj datę).
    JavaScript został stworzony w 1995 roku przez Brendana Eicha w firmie Netscape.
    2)dlaczego jest to język skryptowy
    JavaScript jest językiem skryptowym, ponieważ kod jest interpretowany w czasie wykonywania przez przeglądarkę, a nie kompilowany wcześniej.
    3)gdzie jest wykonywany JS (klient lub serwer)
    JavaScript jest wykonywany po stronie klienta (w przeglądarce), ale może być również używany po stronie serwera (Node.js).
    4)dlaczego jest JS to bezpieczne narzędzie
    JavaScript jest bezpieczny, ponieważ ma ograniczone uprawnienia - nie może bezpośrednio uzyskać dostępu do systemu plików użytkownika ani wykonywać niebezpiecznych operacji.
    5)czy jest to język obiektowy , jeśli tak to dlaczego.
    Tak, JavaScript jest językiem obiektowym, ponieważ obsługuje obiekty, klasy, dziedziczenie i inne cechy programowania obiektowego.

    10)Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu
    ‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"> ›
   
        Tworzy element przycisku HTML z nazwą "przycisk", tekstem "Nowa Strona" i obsługą zdarzenia onclick, które wywołuje funkcję WinOpen().
    Uwaga:
    gdy chcesz wyświetlić instrukcję, która nie powinna się wykonywać a tylko wyświetlać masz do
    dyspozycji wyświetlanie znaków, które nie będą interpretowane jako instrukcja czyli > lub <
    znak < możesz wyświetlić jako ‹
    znak > możesz wyświetlić jako ›
    11)Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
    window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
        Otwiera nowe okno przeglądarki z plikiem "obraz.html", nadaje mu nazwę "okienko" i ustawia parametry wyświetlania.
    kliknij tutaj aby uzyskać opis poniższych opcji
    lub https://kursjs.pl/kurs/rozne/okna
    toolbar=no  ukrywa pasek narzędzi przeglądarki
    directories=no,  ukrywa przyciski katalogów
    menubar=no,  ukrywa pasek menu
    height=280,  ustawia wysokość okna na 280 pikseli
    width=160,  ustawia szerokość okna na 160 pikseli
    top=200,  pozycjonuje okno 200 pikseli od góry ekranu
    left=200  pozycjonuje okno 200 pikseli od lewej strony ekranu
    12)Opisz następującą instrukcję window.close()
        Instrukcja window.close() zamyka aktualne okno przeglądarki lub okno popup.
